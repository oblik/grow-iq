## Step 2: Get FREE Testnet Tokens

### For Sui Testnet:
1. Go to Sui Testnet Faucet: https://discord.gg/sui
2. Join the Discord server
3. Go to `#testnet-faucet` channel
4. Type: `!faucet <your-wallet-address>`
5. You'll receive 10 SUI testnet tokens (FREE!)

## Step 3: Connect Wallet to GrowIQ

1. Open GrowIQ: http://localhost:3002
2. Click "Connect Wallet" button
3. Select your wallet from the popup
4. Approve the connection
5. You should see your testnet balance

---

## Step 4: Make a Test Transaction

1. **Find an Active Pool:**
   - Look for fields marked as "active"
   - Check the minimum stake amount

2. **Click "Invest Now":**
   - Enter amount (use minimum amount for testing)
   - Click "Invest Now"

3. **Approve in Wallet:**
   - Your wallet will popup
   - Check the transaction details
   - Gas fee should be minimal (< 0.01 SUI)
   - Click "Approve"

4. **Transaction Complete:**
   - You'll see a success message
   - Transaction ID will be shown
   - Click "View Transaction" to see on explorer

## Troubleshooting

### "Insufficient Balance" Error
- Make sure you got testnet tokens from faucet
- Check you're on testnet network (not mainnet)
- Verify wallet shows balance

### Wallet Not Connecting
1. Check wallet is unlocked
2. Refresh the page
3. Try disconnecting and reconnecting
4. Clear browser cache

### Transaction Failing
- Ensure you have enough for gas (keep 0.1 SUI for gas)
- Check network is set to testnet
- Try with smaller amount

### No Wallet Popup
- Check popup blocker isn't blocking wallet
- Make sure wallet extension is enabled
- Try different browser (Chrome recommended)
